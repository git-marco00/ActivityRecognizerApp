In order for the application to work, be sure to run both client and server under the same network.

STEPS:
1) run ActivityRecognizer_server
2) start > cmd > ipconfig and find the ip of the server machine
3) install ActivityRecognizer on your Android device
4) when the application starts, insert the ip in the textbox and press "connect"
5) if the ip turns green, then everything is set up correctly, and start monitoring will become clickable.
	 If not, restart from point 1.
6) enjoy :)
